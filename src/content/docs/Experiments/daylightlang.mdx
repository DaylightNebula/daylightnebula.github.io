An experimental programming language using LLVM inspired by Kotlin and Rust.

<img src="/daylightlang.png" />

A basic hello world example looks like this:
```kotlin
const helloWorld: str = "Hello World!\n"

extern printf(text: str*, ...): i32

fun main() {
    printf(helloWorld)
}
```
